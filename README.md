# 📚 하루 한 권 책 정리 (read-book)

매일 책을 추천하고 읽어주는 시스템. 책장 사진에서 판독한 책들을 **하루에 한 권씩** 정리합니다.

## 정리 규칙
1. 고등학생도 읽을 수 있는 난이도로 작성
2. 비유·예시·스토리텔링을 적절히 활용
3. 어려운 개념은 반드시 쉽게 풀어서 설명
4. 분량은 한글 약 20,000자

## 파이프라인
- **노션**: 정리 전문을 저장 (상위 페이지 「📚 하루 한 권 책 정리」의 하위 페이지)
- **카카오톡**: 서브에이전트가 200자 이내 알림(책 제목 + 핵심 한 줄 + 노션 링크)을 동시 발송
- **자동화**: 매일 정해진 시각에 트리거가 다음 대기 책 1권을 자동 정리

## 진행 현황
정리 순서와 상태는 [`reading-list.md`](./reading-list.md) 참고.

목록의 모든 책을 다 읽으면, 읽은 책들이 참고·인용한 책을 목록에 추가해 이어서 진행합니다.
